### ZcrlPortal - простая CMS для больницы.
##### Возможности:
+ Публикация новостей и статей.
+ Публикация истории больницы (отдельная страница на навигационной панели).
+ Публикация всей остальной информации (отдельная страница на навигационной панели).
+ Список персонала больницы (зарегистриванных пользователей системы). Присутствует возможность отбора тех, кто будет отображатся на странице персонал, а кто - нет.
+ Публикация информации по проведённым госзакупкам больницы.
+ Возможности примитивного файлообменника (можно загружать файлы на сайт :) ).
+ Публикация информации об остатках медикаментов (необходима загрузка библиотеки ZcrlMedicamentModels.dll в папку bin, скачать которую можно из репозитория MedicamentRemains).
+ Регистрация пользователя требует подтверждения администратором системы.
+ Логи действий пользователей системы. Каждая пользователь видит совершённые действия других пользователей своей группы. Полезно когда нужно найти того, кто удалил или изменил тот или инной файл, статью, новость.
+ 4 группы пользователей:
  + Обычные пользователи - обычный пользователь, чью регистрацию подтвердил администратор. Имеет собственную страницу, доступную по ссылке. По-умолчанию не отображается в списке персонала.
  + Авторы статей - пользователи, которые могут создавать статьи в разделе Статьи, создавать категории статей. Возможно редактирование и удаление только собственных статей.
  + Редакторы - публикуют и удаляют новости, статьи, редактируют разделы Информация и История.
  + Тендерный комитет - публикация и удаление информации в разделе по госзакупкам.
  + Администратор - неогрнаиченые права: создание и удаление пользователей, информации в любых разделах и т.д.
